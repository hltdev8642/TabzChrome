Claude Code

Essential Flags

Flag Description Example
--dangerously-skip-permissions Skip all permission prompts (use with caution) claude --dangerously-skip-permissions
-p, --print Print response and exit (non-interactive) claude -p "What is 2+2"
--model MODEL Select model (sonnet, opus, haiku) claude --model opus
--resume SESSION_ID Resume a previous session claude --resume abc123
--continue Continue most recent conversation claude --continue

Agent Modes

Agent Examples 
# Explore codebase (read-only, fast)
$ claude --agent explore

# Plan implementation (design-focused)
$ claude --agent plan

# Review code changes
$ claude --agent code-reviewer

# General purpose
$ claude --agent general-purpose

Voice & Audio

Flag Description
--voice NAME Enable voice output with specified voice
--speed RATE Speech speed (0.5 to 2.0)
Voice Examples 
# With voice enabled
$ claude --voice jenny

# Faster speech
$ claude --voice guy --speed 1.5

Useful Slash Commands

Command Description
/help Show help information
/clear Clear conversation history
/config View/edit configuration
/cost Show session cost
/compact Toggle compact output mode
/memory Manage project memory (CLAUDE.md)

Environment Variables

Variable Description
ANTHROPIC_API_KEY API key for authentication
CLAUDE_MODEL Default model to use
CLAUDE_AUDIO Enable audio announcements (1/0)
CLAUDE_VOICE Default voice for TTS

Gemini CLI

Gemini CLI Flags

Flag Description Example
-m, --model MODEL Model selection gemini -m gemini-2.0-flash
-p, --prompt TEXT Send prompt and exit gemini -p "Explain this"
-y, --yes Auto-approve tool calls gemini -y
--sandbox Run in Docker sandbox gemini --sandbox

Codex CLI (OpenAI)

Codex CLI Flags

Flag Description Example
-m, --model MODEL Model selection codex -m o3
--approval-mode MODE suggest, auto-edit, full-auto codex --approval-mode full-auto
--full-auto Shortcut for full-auto mode codex --full-auto
-q, --quiet Minimal output codex -q

Codex Models

codex-mini

Fast, cheap

o3

Reasoning

o3-mini

Balanced

o1

Original

Approval Modes

Mode Description
suggest Only suggest changes, no execution
auto-edit Auto-apply file edits, prompt for commands
full-auto Auto-apply all changes (use with caution!)
